# CreoMedia Project Overview

## Project Description
CreoMedia is a modern digital marketing agency website designed to showcase the agency's services, portfolio, client testimonials, and provide easy contact options. The site is built with HTML, CSS, and JavaScript and includes interactive elements such as animated counters and a chatbot widget.

## Project Structure

- **index.html**: Main landing page with marketing content, portfolio highlights, services, testimonials, counters, and contact widgets.
- **about.html**: About page (not detailed here).
- **services.html**: Services details page (not detailed here).
- **contact.html**: Contact form page (not detailed here).
- **analytics.html**: Analytics or tracking related page (not detailed here).
- **404.html**: Custom 404 error page.
- **css/**: Stylesheet folder containing `style.css` which defines the site's visual design, colors, typography, layout, and animations.
- **js/**: JavaScript folder containing `script.js` which manages interactive behaviors such as navigation toggling, scroll effects, fade-in animations, animated counters, and chatbot functionality.
- **images/**: Folder for image assets used throughout the site.
- **.htaccess**: Apache server configuration file.
- **netlify.toml**: Netlify deployment configuration.
- **nginx.conf**: Nginx server configuration.
- **vercel.json**: Vercel deployment configuration.
- **site.webmanifest**: Web manifest file for PWA support.
